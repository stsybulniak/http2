## Setup  server

```bash
# Run docker-compose
$ docker-compose up

# Import data from data.json file to mongo container
$ npm run import

# Install dependencies
$ npm install

# Run HTTP2 server on port 3000
$ npm run dev
```

## Setup  client

```bash
# Move to client directory
$ cd client

# Install dependencies
$ npm install

# Run client on port 5173
$ npm run dev
```
